# Objetivo

Utilizar o NotebookLM para criar um guia prático para iniciante, começar a investir.
Os prompts são perguntas simples, como se fosse um iniciante aprendendo do zero a realizar os primeiros investimentos.

## Prompts

-> Como começar a investir tendo apenas R$50 por mês?
-> Como montar uma reserva de emergência ganhando pouco?
-> Qual valor devo separar do meu salário para reserva de emergência?
-> Como lidar com as emoções para não tomar decisões impulsivas?
-> Qual a diferença prática entre renda fixa e renda variável?

Através das fontes o NotebookLM gerou um infográfico resultando em um guia prático, uma apresentação em slide resultando em uma Aula sobre finanças pessoais e um relatório que resultou em um post para blog
